# The Internet – Selenium Automation Framework

Automation of https://the-internet.herokuapp.com/ using Python + Selenium + Pytest.

### ✅ Scenarios Automated
1. Form Authentication (valid/invalid login)
2. Add/Remove Elements
3. Dropdown selection
4. File Upload
5. JavaScript Alerts
